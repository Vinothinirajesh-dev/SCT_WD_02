# Stopwatch Web Application

This is a simple stopwatch web application created using HTML, CSS, and JavaScript.

## About the Project
The stopwatch allows users to:
- Start the timer
- Stop the timer
- Reset the timer

This project was created as part of a beginner web development task to practice basic JavaScript functionality.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the project files.
2. Open the folder in VS Code.
3. Open the `index.html` file in any web browser.
4. Click the Start, Stop, and Reset buttons to use the stopwatch.
